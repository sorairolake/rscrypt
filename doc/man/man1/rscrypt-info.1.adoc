== NAME

rscrypt-info - provides information about the encryption parameters

== SYNOPSIS

*rscrypt info* [_OPTION_]... _FILE_

== DESCRIPTION

This command provides information about the encryption parameters.

== POSITIONAL ARGUMENTS

_FILE_::

  Input file.
  If "-" is specified, data will be read from stdin.

== OPTIONS

ifdef::cbor,json,msgpack,toml,yaml[]
*-f*, *--format* _FORMAT_::

  Output format.

  The possible values are:{blank}:::

ifdef::cbor[]
    *cbor*::::

      Concise Binary Object Representation.
endif::cbor[]

ifdef::json[]
    *json*::::

      JavaScript Object Notation.
endif::json[]

ifdef::msgpack[]
    *msgpack*::::

      MessagePack.
endif::msgpack[]

ifdef::toml[]
    *toml*::::

      Tom's Obvious Minimal Language.
endif::toml[]

ifdef::yaml[]
    *yaml*::::

      YAML.
endif::yaml[]
endif::[]

*-h*, *--help*::

  Print help information.

*-V*, *--version*::

  Print version information.
